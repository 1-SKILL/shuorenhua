# 说人话 (shuorenhua)

> AI痕迹消除 · 论文降重 · 文本人性化改写。查重率从60%降到30%以内。

**说人话** 把 AI 生成的文本改写成自然流畅的人类文风。内置 24 条中文 AI 写作特征识别库，每 2-3 天同步最新检测算法。

## Install

```bash
npx skills add 1-SKILL/shuorenhua -g -y
```

## 解决什么问题

- 论文用 AI 写，查重 62%，导师一眼看穿
- 工作汇报像 ChatGPT 写的，领导读两段就问
- 小红书/朋友圈文案，朋友评论"一股 AI 味"

## 效果

| 指标 | 改写前 | 改写后 |
|------|-------|-------|
| 查重率 | 62% | 28% |
| AI 检测 | 高度疑似 | 未检测到明显特征 |
| 语气 | 客观冰冷 | 有温度有判断 |
